💻 Portfólio Kyu (Frontend)
Descrição
Landing page de portfólio profissional desenvolvida com foco em alta performance e design responsivo. O projeto integra animações fluidas e um sistema de captura de leads.

🛠️ Tecnologias
React 19 + TypeScript: Tipagem estática e componentes modernos.

Vite: Bundler de alta velocidade.

Tailwind CSS: Estilização utilitária e responsiva.

Framer Motion: Animações de interface.

Lucide React: Biblioteca de ícones.

React Hook Form + Zod: Validação de formulários.

🚀 Desafios Superados
Configuração de Módulos Estáticos: Implementação de declarações globais (vite-env.d.ts) para suporte a arquivos .png e .svg em ambiente TypeScript.

Path Aliasing: Configuração de caminhos absolutos (@/*) no tsconfig.json para melhor organização do código.
